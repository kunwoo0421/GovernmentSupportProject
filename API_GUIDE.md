
# 🏛️ 공공데이터포털(Data.go.kr) API 키 발급 가이드

정부지원사업 데이터를 실시간으로 가져오기 위해 **API 인증키(Key)**가 필요합니다. 
아래 절차를 따라 키를 발급받아 주세요. (약 5~10분 소요 / 비용 무료)

---

## 1️⃣ 포털 접속 및 로그인
1. [공공데이터포털 (data.go.kr)](https://www.data.go.kr) 접속
2. 회원가입 후 **로그인** 하세요.

---

## 2️⃣ 데이터 찾기
1. 검색창에 **`중소벤처기업부 지원사업정보`** 검색
   - (혹은 `기업마당`으로 검색해도 나옵니다.)
2. 검색 결과 중 **"오픈API"** 탭 클릭
3. **"[중소벤처기업부_지원사업정보]"** (또는 유사한 이름) 클릭
   - *팁: 활용신청 건수가 가장 많은 것을 고르시면 됩니다.*

---

## 3️⃣ 활용 신청 (가장 중요)
1. 파란색 **[활용신청]** 버튼 클릭
2. 신청 양식 작성:
   - **활용목적**: `웹사이트 개발` 또는 `앱 개발` 선택
   - **활용내용 (상세)**: `중소기업 및 소상공인에게 정부지원사업 공고를 쉽게 안내하는 서비스 개발` (간단히 적으셔도 됩니다)
   - **시스템 유형**: `WEB` (웹사이트)
3. **[동의합니다]** 체크 후 **[신청]** 버튼 클릭

---

## 4️⃣ 키(Key) 확인 및 복사
1. 신청 즉시 **"승인"** 됩니다. (자동 승인)
2. **[마이페이지] > [데이터활용] > [OpenAPI 활용현황]** 메뉴 이동
3. 방금 신청한 **"중소벤처기업부_지원사업정보"** 클릭
4. **일반 인증키 (Decoding)** 이라고 된 긴 문자열을 복사하세요.
   - *팁: Encoding 키와 Decoding 키 두 가지가 있는데, 만약 하나가 안 되면 다른 하나를 써야 할 수도 있습니다. 일단 둘 다 복사해두세요.*

---

## 5️⃣ 저에게 알려주기
발급받은 **Decoding 인증키(Key)**를 채팅창에 붙여넣어 주세요. 
제가 바로 코드에 적용하여 **실제 데이터**가 나오도록 연결해 드리겠습니다! 🚀
